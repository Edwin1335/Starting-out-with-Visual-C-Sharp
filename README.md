Event handlers respond to specific events that take place while an
application is running.

When you have a project open in Visual Studio, the time during which you build the GUI
and write the application’s code is referred to as design time. The time during which an 
application is executing is referred to as run time. 

A Label control displays text on a form. Label controls have various
properties that affect the control’s appearance. Label controls can be used
to display unchanging text, or program output.
